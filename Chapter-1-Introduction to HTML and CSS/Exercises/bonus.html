html and css quick guide

a basic guide of html and css

css properties

color

changes text color

p {
  color: red;
}

background-color

sets background color of something

body {
  background-color: #f0f0f0;
}

font-size

controls how big the text is

h1 {
  font-size: 32px;
}

font-family

sets what font is used

body {
  font-family: Arial, sans-serif;
}

padding

adds space inside an element

div {
  padding: 20px;
}

margin

adds space outside an element

p {
  margin: 10px;
}

border

adds a border around stuff

img {
  border: 2px solid black;
}

text-align

lines up your text left, right, or center

h1 {
  text-align: center;
}

html tags

<strong> & <em>

<strong> makes text bold, <em> makes it italic

<strong>bold text</strong>
<em>italic text</em>

<br>

breaks a line (kind of like pressing enter)

line one<br>line two

<p>this is a paragraph</p>

<img>

displays an image on your page

<img src="image.jpg" alt="description">

<ul> <ol> <li>

unordered (bullet points) and ordered (numbered) lists

<ul>
  <li>item 1</li>
  <li>item 2</li>
</ul>

<a>

makes a link to another page or website

<a href="https://example.com">click here</a>

<div>

a container used to group elements

<div>stuff inside here</div>

<input>

used in forms to get user input

<input type="text" placeholder="type here">

<h1> to <h6>

used to make headings from biggest (h1) to smallest (h6)

<h1>this is a heading</h1>

<button>

creates a clickable button

<button>click me</button>
